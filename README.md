# CSCI-4610 Development Environment

This repository provides the **standard Docker + VS Code development environment**
for **CSCI-4610/5610 Concurrent and Parallel Programming**.

All programming in this course is done **inside a Linux Docker container**
using **GCC** (and later OpenMP), regardless of your laptop operating system.

---

## How this repository is used

- This repository is used as a **template** for GitHub Classroom assignments.
- When you accept an assignment, GitHub creates a **private copy** for you.
- You will work inside that private repository.
